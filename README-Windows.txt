TVLK Financial Forecasting Tool
Windows Package

How to run:
1. Extract the zip to a normal folder.
2. Double-click "Launch TVLK Forecasting Tool.bat".
3. Wait a few seconds for the Streamlit app to start in your browser.

Notes:
- Keep all files from the extracted folder together.
- The first launch can take a little longer while Windows warms up the app.
- If Windows Defender prompts you, choose the option that allows the app to run.
